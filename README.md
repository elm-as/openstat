## 🎯 Vision
Pipeline d'automatisation pour le diagnostic et le nettoyage de données statistiques.

> "Un moteur de traitement de données capable d'ingérer des formats hétérogènes (CSV, XLSX, JSON). Il automatise la phase critique de 'Data Cleaning' en identifiant les types de distributions et en signalant les anomalies structurelles avant analyse."
